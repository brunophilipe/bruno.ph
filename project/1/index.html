Noto

Plain-text editor for macOS written in Swift.

macOS Cocoa Swift

A plain-text editor for macOS with a focus on a minimalistic user interface and powerful themes functionality.

Because some people prefer light text on dark background, others dark text on white backgrounds, some others enjoy sepia tones, and others something completely different, it seemed an important feature to allow the user to pick whichever colors they want for the entire document interface.

This project was inspired by a previous project of mine, TipTyper, which was also a plain-text editor, but written in Objective-C. That was a project that taught me Cocoa programming. Now that I feel I have a good knowledge of that library, it felt appropriate to re-do that project, but now using the fantastic Swift programming language.
